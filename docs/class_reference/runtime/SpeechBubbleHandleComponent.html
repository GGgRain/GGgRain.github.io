SpeechBubbleHandleComponent

[[스피치버블 위젯 핸들드]] 클래스와 함께 사용되는 핸들 컴포넌트입니다. 액터에 부착하여 다이얼로그의 참여자를 표현할 때 사용하는 컴포넌트입니다.

[[다이얼로그 위젯 베이스]]에서 사용되는 이벤트나 델리게이트에서 참여자의 레퍼런스는 보통 이 스피치버블 핸들 컴포넌트의 레퍼런스로 주어지며, 이 컴포넌트의 레퍼런스를 이용해 주인 액터에 접근하여 원하는 액션을 가할 수 있습니다.

[[심플 다이얼로그 서브시스템]]의 GetHandleComponent 함수는 월드내에 존재하는 이 클래스의 컴포넌트 객체를 가져오는 함수입니다

프로퍼티​

이름 및 식별자 관련​

이름타입설명
IDNameFName이 컴포넌트가 다이얼로그에서 사용할 참여자ID로, 일종의 식별자로 사용됩니다. 어떤 다이얼로그를 실행 했을 때 그 다이얼로그에 이 IDName이 참가자 ID로 포함이 되어있다면 그 다이얼로그는 이 스피치버블 핸들 컴포넌트의 레퍼런스를 수집하여 저장합니다.
DisplayNameFText다이얼로그 위젯 상에서 유저들에게 보여질 디스플레이용 이름입니다.

다이얼로그 위젯 렌더링 로케이션 관련​

이름타입설명
WidgetOffsetFVector이 컴포넌트가 Speaker(발화자)로 대화에 참여하고 있을 때 스피치 버블 위젯이 렌더링 될 로케이션에 적용할 월드 스페이스 오프셋입니다.
WidgetAttachTargetComponentNameFName스피치 버블 위젯이 렌더링 로케이션의 기준으로 사용할 컴포넌트의 이름입니다. 보통은 머리 위치에 소켓을 만들어서 빈 씬 컴포넌트를 붙혀서 사용하는 것이 권장 됩니다. 지정되지 않으면 컴포넌트의 주인 액터의 루트 컴포넌트 로케이션이 사용됩니다.
